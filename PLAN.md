## Page Structure

### Homepage
- Hero: "Find the Perfect AI Tool"
- Search bar
- Featured tools (sponsors)
- Categories grid
- New tools section
- Newsletter signup

### Category Page
- Category description
- Tool cards (grid)
- Filter by: Free/Paid, Use case, Rating
- Comparison table

### Tool Page
- Tool name & logo
- Description
- Key features
- Pricing
- Pros/Cons
- User ratings
- Affiliate link (CTA)
- Related tools
